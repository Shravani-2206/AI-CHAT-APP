# 🤖 AI Chat App

## 📌 Description

AI Chat App is a simple web-based chatbot application built using HTML, CSS, and JavaScript.
It allows users to ask questions and receive intelligent responses using the Google Gemini API.

## 🚀 Features

* Interactive chatbot interface
* Real-time AI responses
* Clean and responsive UI
* Simple and user-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Gemini API

## 📂 Project Structure

* `index.html` – Main user interface
* `style.css` – Styling of the app
* `script.js` – Chat logic and API integration

## ⚙️ How It Works

1. User types a message in the input box
2. Message is sent to the Gemini API
3. AI processes the input and returns a response
4. Response is displayed in the chat window

## ▶️ How to Run

1. Clone or download this repository
2. Open `index.html` in your browser
3. Start chatting with the AI

## 🔑 API Setup (IMPORTANT)

* Get your API key from Google AI Studio
* Replace:

  ```js
  const API_KEY = "YOUR_API_KEY_HERE";
  ```
* ⚠️ Do NOT share your API key publicly


## ⚠️ Limitations

* Requires internet connection
* API key needed for responses
* Limited free usage based on API plan
